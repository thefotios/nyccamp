Infrastructure as a Service

(IaaS)


Servers run in the cloud

You build and manage EVERYTHING


Guess how much time you actually spend on your application?

Software as a Service

(SaaS)


Somebody else's application running in the cloud

You're restricted to the features of their application


Want to change something?

Good luck with that...

Platform as a Service

(PaaS)


Your applications

Our platform

Running ANYWHERE

Much more flexible than SaaS

CLI Example

Create

rhc app create -a myapp -t jbossas-7

Code

cd myapp
cp /path/to/war/myfile.war ./deployments

Deploy

git add ./deployments/myfile.war
git commit -m "OpenShift is easy"
git push
