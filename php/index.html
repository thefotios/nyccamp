So What is PaaS?



It's kind of like Amazon, right?
Nope!

IaaS


Infrastructure as a Service


Servers run in the cloud


You build and manage EVERYTHING
(OS, App Servers, Database)


Guess how much time you actually get to spend on your application?

SAAS


Software as a Service


Somebody else's application running in the cloud


You're restricted to the features of their application


Want to change something? Good luck with that...

Paas

Platform as a Service

Your applications
Our platform
Running ANYWHERE

Much less work than IaaS
Much more flexible than SaaS



To put it all together



Why Paas?


YOU code the application, WE run it for you

Leverage the ease, scale, and power of the Cloud


Code
Deploy
Enjoy!
